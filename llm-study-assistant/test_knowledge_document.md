# Python编程基础测试文档

## Python语言特性

Python是一门解释型语言，具有以下特性：
- 开源：Python是开源软件，可以自由使用和修改
- 跨平台：Python可以在多种操作系统上运行，包括Windows、macOS和Linux
- 面向对象：Python支持面向对象编程范式
- 解释型：Python是解释型语言，不需要编译成机器码

## Python保留字

Python有一些保留字（关键字），不能用作变量名或标识符：
- class：用于定义类
- def：用于定义函数
- for：用于循环语句
- if, else, elif：用于条件语句
- while：用于循环语句
- import：用于导入模块
- from：用于从模块中导入特定部分
- return：用于从函数返回值

以下不是Python的保留字：
- main：这不是Python的保留字

## Python数据类型

Python支持多种内置数据类型：
- int：整数类型
- float：浮点数类型
- string：字符串类型
- boolean：布尔类型（True/False）
- list：列表类型
- tuple：元组类型
- dict：字典类型
- set：集合类型

## 列表与元组的区别

列表和元组都是Python中的序列类型，但有重要区别：
1. 可变性：
   - 列表是可变的，可以添加、删除和修改元素
   - 元组是不可变的，一旦创建就不能改变
2. 语法表示：
   - 列表使用方括号[]表示
   - 元组使用圆括号()表示
3. 性能：
   - 元组比列表占用更少的内存
   - 元组的遍历速度比列表快
4. 用途：
   - 列表用于需要修改数据的场景
   - 元组用于不需要修改的数据，如坐标点、数据库记录等
