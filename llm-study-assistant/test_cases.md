# 视频分析功能测试用例集

## 1. 成功案例

### 1.1 基本选择题
**输入文本:**
```
1. 下列关于Python的说法，错误的是？
   A. Python是开源的 B. Python是跨平台的 C. Python是面向对象的 D. Python是编译型语言
```

**预期输出:**
- 问题: 下列关于Python的说法，错误的是？
- 选项: ["A. Python是开源的", "B. Python是跨平台的", "C. Python是面向对象的", "D. Python是编译型语言"]
- 答案: D. Python是编译型语言（Python是解释型语言）

### 1.2 多种题号格式
**输入文本:**
```
2、哪个选项不是Python的保留字？
   A) class   B) def   C) for   D) main
```

**预期输出:**
- 问题: 哪个选项不是Python的保留字？
- 选项: ["A) class", "B) def", "C) for", "D) main"]
- 答案: D) main（main不是Python的保留字）

## 2. 边缘案例

```

2、哪个选项不是Python的保留字？
```

**预期输出:**
- 应该正确解析出3个独立的问答对

## 3. 失败案例收集

### 3.1 无法识别的格式
**输入文本:**
```
问题1：下列关于Python的说法，错误的是？
选项：A. Python是开源的 B. Python是跨平台的 C. Python是面向对象的 D. Python是编译型语言
```

**问题:**
- 当前的正则表达式可能无法正确识别这种格式

### 3.2 特殊字符干扰
**输入文本:**
```
1. 下列关于Python的说法，错误的是？*****
   A. Python是开源的 B. Python是跨平台的 C. Python是面向对象的 D. Python是编译型语言
```

**问题:**
- 特殊字符可能会干扰OCR文本的解析
